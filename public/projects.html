Super Robot Wars

GitHub Deployed App


Notes:
This app was built for my final project at the UT Coding Bootcamp with 2 other cohorts. It is a multiplayer tactical role-playing game based off of the Bandai series Super Robot Wars. I was in charge of building the gameplay UI on the front-end and intergrating socket.io so that the various clients playing would remained synced. Since games of this genre are typically designed to be single player games, I assisted our group's game engine developer with much of the game design to ensure that gameplay was fair.

Tech Used:
jQuery, jQuery UI, Node.js, Express.js, MongoDB, Mongoose, Socket.io, Embeddedjs, Passport, Bootstrap

Gig.ly

GitHub Deployed App


Notes:
Gig.ly was my second group project at the UT Coding Bootcamp. Our group was randomly assigned so delegating tasks and cooperating with my groupmates was crucial and I believe we executed well on these fronts. I was the front-end engineer tasked with building and designing all of the UI for the app as well as integrating Fullcalendar.io. Gig.ly is a scheduling app for employers to schedule and hire for short-term gigs. The app allows users to create an admin account that can schedule gigs and provides a key that employees can use to sign in and schedule gigs for themselves. As soon as an employee submits a "gig-it" request, employers will recieve a text notification with employee's contact info and the gig that was scheduled.

Tech Used:
jQuery, Node.js, Express.js, MySQL, Sequelize, Fullcalendar.io, Moment.js, Twilio, Bulma.io
